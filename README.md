# G-COB — Giordano Cobrança e Assessoria

Landing page institucional da G-COB, empresa especializada em recuperação de créditos e cobrança extrajudicial.

## Sobre

Site estático hospedado via GitHub Pages, acessível em [gcob.com.br](https://gcob.com.br).

## Tecnologias

- HTML5 + CSS (Tailwind CDN)
- Google Fonts (Playfair Display + IBM Plex Sans)
- Formspree (formulário de contato)

## Estrutura

```
gcob-site/
├── index.html   # Site completo (HTML, CSS, JS)
└── CNAME        # Domínio customizado gcob.com.br
```

## Deploy

O site é publicado automaticamente via GitHub Pages a partir da branch `main`.

Qualquer push na branch `main` atualiza o site em produção em instantes.
